Braille Label Generator

Generate braille labels for 3D printing or other uses. Enter text and download the result (e.g. STL).

Generator

Output
Enter text and options, then click Generate STL. A .stl file will be downloaded.

Tips

This browser version generates a single label using the same parameter model as the CLI. For batch generation, use the Node CLI.

  • Enter the text you want in braille; adjust size and options as needed.
  • Export as STL for 3D printing or other formats if supported.

Printing

  • Layer height: 0.2–0.28 mm.
  • Print with a flat base for stability.
